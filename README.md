## *PROJECT SDA - SISTEMA DE AGENDAMENTO / SCHEDULING SYSTEM* ##
27/04/2026 - 
O objetivo principal do sistema é oferecer uma mecânica funcional para o gerenciamento de agendamentos, possibilitando a organização e a agilização da rotina de agendamento com maior dinâmica e controle. Com uma organização e legitimidade para reduzir conflitos entre horários e a desorganização de agendamento manual, trazendo mais confiança e facilidade para o estabelecimento e os seus clientes.
